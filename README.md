# Auto-Beer

Autamatically uses rootbeer whenever you cast one of the supported skills.

## Supported Skills:   
- Warrior 
  - **Deadly Gamble**  
- Berserker
  -  **Unleash**  
- Sorcerer
  -  **Mana Boost**    
- Valkyrie
  -  **Ragnarok**
  -  **Godsfall**
- Lancer
  -  **Adrenalin Rush**  
- Slayer
  -  **In Cold Blood**     
- Reaper
  -  **Shadow Reaping**    
- Archer
  -  **Windsong**
- Gunner
  -  **Modular Weapon System**
- Ninja
  -  **Inner Harmony**


## Commands 

- **(Default mode: ab2)**
- *ab0*
  -  **Disabled the module**   
- *ab1*
  -  **Enables auto brooch only**  
- *ab2*
  -  **Enables auto beer only**    
- *ab3*
  -  **Enabled auto brooch and beer**
